== Description ==
这是一款付费阅读插件，可以为你的文章设置费用，每个用户都需要付费阅读你的文章。

这个插件依赖于 WordPress 自身的用户需要，使用时需要开启允许所有人注册站点的功能。

插件的支付网关为 payjs.cn ，你需要注册该站点以获取设置时所需要的密码。

== Installation ==

1. upload the plugin to `wp-content/plugins/pay-to-view-all`,or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Settings->付费阅读设置 screen to configure the plugin
4. Set All user can register your website.
